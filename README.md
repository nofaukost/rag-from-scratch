# RAG from Scratch

Learn how to implement RAG (Retrieval Augmented Generation) from scratch, straight from a LangChain software engineer. This Python course teaches you how to use RAG to combine your own custom data with the power of Large Language Models (LLMs). [Watch at youtube](https://www.youtube.com/watch?v=sVcwVQRHIc8)
